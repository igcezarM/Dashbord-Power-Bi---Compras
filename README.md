Dashboard de compras desenvolvido para monitorar custos, eficiência operacional e desempenho de fornecedores, apoiando a tomada de decisão estratégica.

O projeto resolve problemas como falta de controle de gastos, baixa visibilidade dos processos, atrasos nas entregas e ausência de indicadores de performance, por meio de KPIs como Custo Total, Lead Time de Compra, Atraso na Entrega, Lead Time de Fornecedor e Savings.

Como resultado, o dashboard proporciona maior controle financeiro, identificação de gargalos e otimização das decisões na área de compras. 🚀
